Usage: 

    Select item:
    - `1`: Shoulder
    - `2`: LowerArm
    - `3`: Elbow
    - `4`: UpperArm
    - `5`: Wrist

    Move Item:
    `Left && Right`: to rotate

    Move Camera
    - `WASD`: horizontal movment
    - `LShift`: move down
    - `Space`: move up
    - `ESC`: to grab/release cursor
